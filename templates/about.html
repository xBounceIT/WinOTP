About WinOTP

WinOTP

Version 1.0.0

A desktop TOTP (Time-based One-Time Password) authenticator for Windows

Features:
  • Generate TOTP codes for two-factor authentication
  • Add tokens via QR code or manual entry
  • One-click copy to clipboard
  • Import and export tokens for backup
Credits

WinOTP uses the following open-source libraries:

  • Flask - Web framework
  • PyWebView - Desktop window wrapper
  • PyOTP - TOTP generation
  • Bootstrap - UI framework
  • jQuery - JavaScript library
  • PyZBar - QR code scanning
