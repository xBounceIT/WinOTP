About WinOTP

WinOTP Authenticator

A simple, secure TOTP authenticator for Windows with Windows Hello integration.

Version: 1.0.0

Features

  • Generate TOTP codes for two-factor authentication
  • Scan QR codes to add tokens
  • Manual entry of token details
  • NTP time synchronization for accurate codes
  • Windows Hello integration for secure access
